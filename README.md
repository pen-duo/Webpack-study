# Webpack-study

## 安装 webpack

> 全局安装：cnpm i webpack webpack-cli -g  
> 本地安装：cnpm i webpack webpack-cli -D

## 安装依赖

> node_modules安装在最外面，在最外层运行 `cnpm i` 本地下载安装loader和plugins
